# Installation

## Prerequisites

- [Bun](https://bun.sh) v1.3.10 or later
- [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) CLI installed
- A GitHub or Asana account with API access

## Install from npm

```bash [Terminal]
bunx @pleaseai/agent init
```

The `init` command will:

1. Create a GitHub Project (or select an existing one)
2. Generate a `WORKFLOW.md` file in your repository
3. Set up the required labels on your project

## Manual Setup

If you prefer to set up manually:

### 1. Install the package

```bash [Terminal]
bun add -g @pleaseai/agent
```

### 2. Create WORKFLOW.md

Create a `WORKFLOW.md` file in the root of your target repository:

```markdown [WORKFLOW.md]
---
tracker:
  type: github_projects
  api_key: $GITHUB_TOKEN
  project_url: https://github.com/orgs/YOUR_ORG/projects/1

polling:
  interval_ms: 30000

workspace:
  root: ./workspaces

agent:
  max_concurrent_agents: 3
  max_turns: 10
---

You are working on issue {{ issue.title }}.

{{ issue.body }}
```

### 3. Set environment variables

```bash [Terminal]
export GITHUB_TOKEN=ghp_your_token_here
```
