# Chat Bot Integration

Agent Please includes a multi-platform chat bot built with the [Chat SDK](https://github.com/nicepkg/chat). When users `@mention` the bot, it responds with current orchestrator status.

## Supported Platforms

### GitHub Issue Comments

The GitHub adapter handles `@mention` in issue comments via the webhook endpoint.

**Requirements:**
- GitHub Projects v2 tracker configured
- Webhook secret set

The bot activates automatically when GitHub tracker config is present. No additional configuration needed.

### Slack

**Requirements:**
- `SLACK_BOT_TOKEN` environment variable
- `SLACK_SIGNING_SECRET` environment variable

```bash [Terminal]
export SLACK_BOT_TOKEN=xoxb-your-token
export SLACK_SIGNING_SECRET=your-signing-secret
```

The Slack adapter handles events via `/api/webhooks/slack`.

## Bot Responses

When mentioned, the bot responds with:

- **Running issues** — currently active agent sessions
- **Retry queue** — issues waiting for retry
- **Token usage** — cumulative token consumption

## Webhook Endpoints

| Platform | Endpoint |
|----------|----------|
| GitHub | `POST /api/webhooks/github` |
| Slack | `POST /api/webhooks/slack` |

## Lifecycle

The chat bot is managed as a Nitro server plugin. It starts with the server and shuts down gracefully when the service stops.
